## ü§ù How to Contribute

We welcome contributions from all team members! Here's how to get started:

1. **Pick an Issue** - Check our project boards for open issues
2. **Fork & Branch** - Create a feature branch from `main`
3. **Code** - Follow our coding standards and write tests
4. **Test** - Ensure all tests pass locally
5. **Pull Request** - Submit a PR with a clear description
6. **Review** - Address feedback from code reviews

### Coding Standards
- Follow the language-specific style guides
- Write meaningful commit messages
- Add documentation for new features
- Include tests for all changes
- Keep PRs focused and reasonably sized
